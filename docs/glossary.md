# Glossary

## Brain

Allowed only when decomposed into context selection, memory retrieval and
application, source grounding, policy control, review feedback, and pattern
selection.

## Memory

Not markdown notes, source list, or compaction summary. Memory is store-backed,
typed, temporal, source-linked, confidence-aware, invalidatable, and
feedback-aware.

## Context

Not everything relevant. Context is a task-specific working set selected under
budget with non-goals and application guidance.

## Skill

Not a prompt snippet, project-specific ritual, or stack agent. A skill is a
reusable engineering discipline with a clear trigger, lifecycle, verification,
and removal condition.

## Eval

Not proof of product quality. Eval is a regression guard for a real contract,
failure, golden task, or security boundary.

## Source

Not an impressive link. Source is a mechanism carrier with decision, risk, or
rejection support and `doesNotProve`.

## Dashboard

Not product proof. Dashboard is a later action surface over typed objects.

## Subagent

Not another brain. A subagent is a bounded role with separate context and an
output contract, usually read-only/proposal-only.

## Goal

Current objective, constraints, evidence, stop condition, and next action. Not
product brain.
