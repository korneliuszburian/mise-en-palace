# MM-00 Source-To-Decision Ledger

MM-00 retains sources only when they produce a mechanism, KRN implication,
decision or rejection, consumer, does-not-prove statement, falsifier, and next
slice.

Raw local materials remain quarry in MM-00. They are not promoted into runtime
truth and are not Memory Core.

## MM-00 Adoption Rule

No source is retained because it is impressive. A source survives only if it
changes a KRN decision, rejects a path, defines a risk, creates a testable
hypothesis, or constrains a future slice.
